# MULTI-GNN For Lung Cancer Biomarkers

This project uses the power of Graph Neural Networks to find the main biomarkers for Lung Cancer.
It combines gene expressions and methylation data from the Illumina 27 dataset with multiple evaluation techniques, such as Louvain, Lasso, SVM, PCA and KBest.
